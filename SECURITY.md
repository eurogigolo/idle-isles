# Security

Idle Isles includes Solidity contracts and wallet-connected frontend code. Treat reports about
asset loss, unauthorized minting, marketplace escrow, claim settlement, randomness, or deployment
configuration as security-sensitive.

## Reporting

Do not publish exploitable contract or wallet issues publicly before maintainers have a chance to
respond. If this repository is hosted on GitHub, use GitHub private vulnerability reporting when
available. Otherwise, contact the repository owner through their preferred private channel.

Include:

- Affected commit or deployment address.
- Network and chain ID.
- Steps to reproduce.
- Expected and actual behavior.
- Whether funds or tradable assets are at risk.

## Current Status

The contracts are unaudited and intended for prototype/testnet use until the production readiness
checklist is complete.
